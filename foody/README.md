# 🍜 Foody（Beta）— 大马美食分享平台

一个让大马用户宣传、分享美食的小天地。**免费、无广告、不涉及任何金钱功能。**

---

## 怎么启动

**方法一（最简单）：** 双击 `start-foody.bat`，等 2 秒浏览器会自动打开。

**方法二（命令行）：**

```
cd foody
npm install   （只有第一次需要）
npm start
```

启动后打开 → **http://localhost:3000**

> 停止服务器：在黑色窗口里按 `Ctrl + C`，或直接关掉窗口。

## 用手机访问（同一个 WiFi）

服务器启动时，黑色窗口会显示一行 `手机(同一WiFi): http://192.168.x.x:3000`。
在手机浏览器输入那个网址，就可以直接用手机注册、从相册发布美食。

## 示范账号

第一次启动会自动建立 3 个示范帖子（插画），方便你看效果：

| 用户名 | 密码 |
|---|---|
| `foody_demo` | `foody123` |

登录这个账号后，可以把示范帖子逐个删除（帖子右侧的垃圾桶按钮）。

## 功能一览

- **首页**：注册（用户名 + 密码 + 电话 + 地区）、登录、自动登入（180 天内免重新登录）
- **三语**：中文 / Bahasa Melayu / English，右上角随时切换
- **FYP**：TikTok 式全屏上下滑 feed
  - 每个帖子：点赞 ❤️、留言 💬、收藏 🔖、**WhatsApp 一键联系发布者** 💚
  - 视频：音量 0–100%、亮度 0–100%、静音（右上角滑杆图标）
  - 双击画面也可以点赞；键盘 ↑↓ 也能翻帖子
  - 按州属筛选 / 查看我的收藏
- **发布**：手机相册或电脑选择照片/视频（最大 150MB），必须选择地区
- 游客可以浏览，但点赞/留言/收藏/发布/WhatsApp 一律需要账号（符合 Beta 规则）

## 数据放在哪？

| 内容 | 位置 |
|---|---|
| 账号、帖子、留言、点赞… | `data/db.json` |
| 用户上传的照片/视频 | `uploads/` 文件夹 |

**备份**：把 `data` 和 `uploads` 两个文件夹复制一份就行。

## 小提示

- 换端口：先设置环境变量再启动，例如 `set PORT=3001 && npm start`
- 视频格式建议用 **MP4**（所有浏览器都支持）；iPhone 的 .mov 多数浏览器也可以播放
- 这是 Beta 版本，密码已加密存储（scrypt），但正式上线前建议再加 HTTPS
- 以后想放上互联网（让所有人访问），可以再部署到云端——到时候再说 😄
